# 11. Profiling

Agregar el profile a la ruta /TEST
```js
logger.profile("test", { level: "info" });

// inicia el contador
const profiler = logger.startTimer();

setTimeout(() => {
  // finaliza el contador y registra duración
  profiler.done({ message: "Logging message" });
}, 5000);
```
